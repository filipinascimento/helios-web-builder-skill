# __APP_TITLE__

Standalone Vite viewer using `helios-web` and `helios-network`.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The starter creates a small in-browser demo network. Replace `createDemoNetwork()` in `src/main.js` with a loader for your `.xnet`, `.zxnet.json`, `.gt.zst`, or generated data.
